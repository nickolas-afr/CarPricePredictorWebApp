# Dataset Instructions

To train the ML model, you need to download the Cars Germany dataset from Kaggle.

## Download Instructions

1. Visit: https://www.kaggle.com/datasets/ander289386/cars-germany
2. Download the dataset
3. Extract the file `autoscout24-germany-dataset.csv`
4. Place it in this folder (`CarPricePredictor.ML/Data/`)

## File Requirements

- File name: `autoscout24-germany-dataset.csv`
- Location: `CarPricePredictor.ML/Data/autoscout24-germany-dataset.csv`

Once the file is in place, you can run the `CarPricePredictor.ML` console application to train the model.

## Note

The CSV file is not included in the repository due to its size. You must download it manually from Kaggle.
